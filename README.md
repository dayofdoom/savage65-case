# savage65-case
This stacked acrylic case was inspired by the Rama Works Thermal case. 

* Acrylic width should be 3mm (all layers except plate)
* Plate should be 1.5mm POM 

PCB Compatibility
* Savage65
* Instant65
* KBD67 Rev2
* Probably any other 65% PCBs with top left connector
